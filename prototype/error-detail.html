Integration by parts problem

Compute the integral: ∫x·ln(x)dx using integration by parts.

Where I Went Wrong

I incorrectly chose u = ln(x) and dv = x dx, which made the resulting integral more complicated than the original. The correct choice should have been u = ln(x) and dv = x dx, but I made an error in computing v after integration.

Correct Solution

Using integration by parts formula: ∫u·dv = u·v - ∫v·du

  • Let u = ln(x) and dv = x dx
  • Then du = (1/x) dx and v = x²/2
  • ∫x·ln(x)dx = ln(x) · (x²/2) - ∫(x²/2) · (1/x) dx
  • = (x²/2)·ln(x) - ∫(x/2) dx
  • = (x²/2)·ln(x) - (x²/4) + C

Final answer: ∫x·ln(x)dx = (x²/2)·ln(x) - (x²/4) + C

Personal Notes

For integration by parts, remember to choose u and dv strategically. A good strategy is LIATE: choose u as the first function that appears in the LIATE order (Logarithmic, Inverse trigonometric, Algebraic, Trigonometric, Exponential), and dv as the remaining factor.
